# Ubuntu Budget App

## 📱 Overview
Ubuntu Budget is a mobile budgeting application that helps users track expenses, manage spending categories, and visualize financial habits.

## 🚀 Features
- User authentication (login/register)
- Add and manage expenses
- Budget categories (envelope system)
- Monthly budget tracking
- Graphical spending analysis
- Digital receipt capture

## 🛠 Technologies Used
- Kotlin
- Android Studio
- Room Database
- Coroutines
- MPAndroidChart
- CameraX
